# ■ 要件定義

## ① アプリの目的・ゴール
・対象ユーザー：AWS SAAを学習する初心者
・解決したいこと：問題を解くだけでなく、AI解説で理解を深められるようにする
・MVPの完成条件：IAMカテゴリの問題を5問解けて、回答後に解説が表示される

## ② 機能要件
やること
・問題を表示できる
・選択式で回答できる
・正解・不正解を判定できる
・回答後にGemini APIを使って解説をリアルタイムで生成・表示できる

やらないこと
・ログイン機能
・学習履歴の保存
・問題の自動生成
・複数カテゴリ対応

## ③ 非機能要件
・対応端末：まずはPCブラウザ
・画面表示：大きな待ち時間なく動くこと
・セキュリティ：個人情報を扱わないので最小限でよい
・API呼び出しに数秒かかることを許容する
・APIエラー時はエラーメッセージを表示する
・APIキーはコードに直書きしない（.envファイルで管理）

## ④ データ設計
解説はGemini APIで都度生成するため、JSONにexplanationは持たない
```json
{
  "id": 1,
  "question": "問題文",
  "choices": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  "answerIndex": 0,
  "category": "IAM"
}
```
※ answerIndexは0始まりとする

## ⑤ 画面設計
必要な画面
・問題表示画面
・回答結果・解説表示画面

ユーザーの操作フロー
1. 問題が表示される
2. 選択肢を選ぶ
3. 回答する
4. 正解・不正解が表示される
5. 「Geminiで解説を生成中...」が表示される
6. 解説が表示される
7. 次の問題へ進む

## ⑥ 技術スタック・実装方針
・言語：Python
・フレームワーク：Streamlit
・データ管理：JSONで管理（DBは使わない）
・AI API：Gemini API（解説生成のみに使用）
・SDK：google-genai（pip install google-genai）
・APIキー管理：.envファイル
・まずはローカルで動けばOK
